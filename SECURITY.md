# Security Policy

## Supported Versions

The latest version of this project is actively maintained and receives security updates.

## Reporting a Vulnerability

If you discover a security vulnerability, please do **not** open a public GitHub issue.

Instead:

1. Contact the project maintainer privately.
2. Include a detailed description of the vulnerability.
3. Provide steps to reproduce the issue, if possible.
4. Allow reasonable time for the issue to be investigated and fixed before public disclosure.

Thank you for helping keep this project secure.
